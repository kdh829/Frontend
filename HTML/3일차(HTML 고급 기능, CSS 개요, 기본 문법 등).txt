// HTML 고급 기능

1. 영상 삽입
<iframe>이 있는 문장을 소스코드 내에 삽입해주면 된다.

2. 댓글 기능 : Disqus나 LiveRe라는 서비스가 있다. 여기서 소스 코드를 복사해서 붙여넣기를 하면되는데,
안 열릴 수가 있다. 이건 file:// 즉. 로컬 웹 브라우저에서 직접 접속했다는 뜻이므로 웹 서버를 켜주고 IP주소+포트 
뒤에 index.html 같이 파일명을 붙여주면 Disqus의 댓글 창에 접속이 가능하다 -> 내 웹에서 다른 사람이 직접 댓글을 달 수 있게 된다.

3. 채팅 기능 : tawk라는 웹 사이트를 통해 채팅 소스를 가져온다. 여기서도 처음엔 연결 오류가 뜨는데, 웹 서버를 거쳐
웹 페이지가 열리도록 도메인 주소를 할당해준다( webserver for chrome )

4. 방문자 추적 기능 : 구글 에널리틱스 사이트에서 소스코드를 가져와 활용할 수 있다.

// CSS 기본 내용

HTML이 문서를 만드는데 활용되고 사람들은 이어서 전세계 사람들과 공유할 수 있게 됐다. 그러다 문득 예쁜 디자인을
갈구하게 되고 새로운 태그를 붙임으로써 다양한 색이나 도형 등을 구현할 수 있게 되는데.. 이때 사용되는 태그 언어가
CSS 이다. HTML은 이것이 어떤 목적에 쓰는 것인지( ex. strong 태그는 '강조'라거나 ) 정해져있지만, CSS는 어떤 문장에
의미를 부여하기 위한 언어는 아니다. '디자인 자체가 정보는 아니다'라는 말로 풀 수 있을 것 같다. 

초창기 웹 브라우저는 맨 처음 코드를 해석할 때 인지했던 언어는 HTML이다. 기본적으로 웹 브라우저는 코드 = HTML.
즉, 디폴트가 HTML이라고 인식하기 때문에 웹 브라우저에게 난 CSS를 쓰겠다고 소스를 통해 알려줘야 한다.

1. 태그 / 코드

1-1) 태그 'style' : HTML의 문법이면서 동시에 안쪽은 모두 CSS 내용들이다. 
1-2) 코드 a : "웹 브라우저야, 모든 <a> 태그에 대해서"라는 뜻을 의미하는 코드는 a이다. 그리고 중괄호를 열고 폰트 색상이
이렇다는 것을 알려줘야 한다 ( ex. <style> a { color: red; } </style> ... 생략 )

1. 자바스크립트란?

1) 사용자와 상호작용하는 언어이다. 다른 말로 HTML을 제어하는 언어이다.

2) HTML은 특성상 웹 브라우저 화면에 한 번 출력되면 자기자신을 바꿀 수 없다. <body>태그는
영원한 <body> 태그이다. 하지만, 자바스크립트를 이용해 버튼을 누르면 코드에 따라 <body>에 
style 속성이 추가되고 <body> 디자인을 바꿀 수 없다.

3) 예시로 <input> 태그로 버튼을 만들 수 있고, onclick 속성으로 사용자 동작에 따른 자바스크립트를
수행할 수 있다. 만약 style 속성 값을 <body> 태그에 주고 싶다면 document.querySelector( 'body' )를
쓰면 된다.

근데 document 하고 . 이 자바스크립트 문법이라는 건 알겠는데 html에서는 어떻게 알게 할까?
-> <script> 태그를 쓰면 된다. <script> 코드 </script> 에서 코드는 자바스크립트 언어임을 의미한다.

+) 자바스크립트가 동적인 언어임은 간단한 로직으로 확인할 수 있다.
1+1이란 것을 html에 작성하고 / <script> 내에 document.write( 1+1 ); 을 작성해서 비교해본다.
그러면 html에는 1+1 이 출력되고 자바스크립트 단에선 2 가 출력되는 것을 확인할 수 있을 것이다.


2. 이벤트

1) 어떤 속성에 의해 자바스크립트 코드가 실행될 때, 해당 소스를 웹 브라우저는 기억하고 있다가,
실제 사용자 동작 발생 시 소스를 실행한다. 즉, 웹 브라우저에서 일어나는 사건의 개념이라고 볼 수 있다.

예를 들어 onclick이라는 속성은 자바스크립트 소스에 대해서만 실행이 된다. input으로 선언한 버튼을
클릭하면 onclick 내에 자바스크립트 구문을 웹 브라우저가 기억하고 있다가 출력한다.

2) 이벤트 종류( on으로 시작한다 )
- onchange : 내용이 변했을 때를 체크한다. 한번 수정하고 Enter 칠 때마다 수행한다. 내용을 수정하지 않으면 실행하지 않는다.
- onkeydown : 최소 하나의 입력 값을 변화를 줄 때마다 수행한다( 이후 수정 내용이 반영된다 )
- 이외 18~28 개 정도의 이벤트가 존재한다. on으로 시작한다는 것만 알아뒀다가 검색하면서 알아가기

4. 데이터 타입( 자료형 )

1) 자바스크립트에는 총 6개의 자료형과 객체가 있다. 대표적으로는 숫자와 문자열이 있다.
1-1) 자료형
- 숫자 : 중요한 부분은 연산이다. 더하기( + )는 이항연산자 이면서 산술 연산자라고도 한다.
- 문자열 : (" "), (' ') 둘 다 가능하다. 

여기서 문자열에 여러 기능들이 하위로 존재한다. 먼저, ' .length '가 있다. 이런 것들을 Properties라고 한다.

' .toUpperCase() ' 은 메소드라고 한다( 'Hello World'.toUpperCase() 
이외 메소드로는..
(1) .indexOf() : 안에 주는 문자 값이 몇 번째 오는 건지 출력해준다. ( ex. 'Hello World'에서 .indexOf('World')를 하면 
-> H를 0번째로 보고 World가 5번째라는 의미로 '5'가 출력된다 )
(2) .trim() : 공백을 빼고 결과 값을 출력한다.
